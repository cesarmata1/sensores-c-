# Sensores-C

Este repositorio contiene una práctica desarrollada en **C** relacionada con el uso de sensores y comunicación mediante sockets. El proyecto forma parte de los ejercicios realizados dentro de la asignatura de programación y sistemas.

## Descripción

El código aquí presente está escrito en **C** y muestra el uso de sockets para manejar datos provenientes de sensores. El objetivo de la práctica es entender cómo trabajar con programación de bajo nivel, entrada/salida y comunicación entre procesos o dispositivos.

## Tecnologías utilizadas

- Lenguaje C
- Sockets para comunicación
